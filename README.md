# LZFD
Landzone Files Fast&amp;Secure Download

Version 1.6 - 10/11/2025
Windows 11&10 x64

Logiciel qui permet de télécharger en sécurité les logiciels du site LandZone.
Vérification du hash et scan avec antivirus de Windows.
Téléchargement ultra rapide avec possibilité de pause, en multitache.
